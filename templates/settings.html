Spotify Data Import

1. Go to spotify.com/account/privacy

2. Scroll to Download your data and check Account data.

3. Click Request data. Spotify will prepare your data (usually takes a few hours to days).

4. You will receive an email. Download the zip and extract it.

5. Select the extracted folder path above. It should contain files like Playlist1.json and YourLibrary.json.
